# 🎯 Jogo do Número Secreto

## 📖 Sobre o Projeto

O **Jogo do Número Secreto** é uma aplicação web desenvolvida como projeto de conclusão de um curso de **Lógica de Programação**.

O objetivo do jogo é desafiar o usuário a descobrir um número secreto gerado aleatoriamente pelo sistema, recebendo dicas a cada tentativa até encontrar a resposta correta.

Neste projeto, atuei principalmente no desenvolvimento da lógica da aplicação utilizando **JavaScript**, implementando regras de negócio, validações e mecanismos de interação responsáveis pelo funcionamento do jogo.

---

## 🚀 Funcionalidades

* Geração aleatória de números secretos.
* Validação das tentativas do jogador.
* Feedback informando se o número secreto é maior ou menor que o valor informado.
* Contagem de tentativas realizadas.
* Reinicialização automática da partida após o acerto.
* Interface simples e intuitiva para interação online.
